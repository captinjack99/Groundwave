# Groundwave vs. HD Radio — an honest, numbers-backed scorecard

*What a modern, royalty-free digital-radio waveform actually buys you over hybrid
IBOC (HD Radio), measured rather than asserted.*

## Premise (and the constraint that shapes it)

The interesting question isn't "could you replace FM with an all-digital channel" —
academically clean, but a non-starter: **no station will surrender its analog signal**
for a digital-only mode no listener can yet receive. That chicken-and-egg is exactly
why HD Radio runs in *hybrid* mode, bolting digital sidebands onto the existing analog
carrier. So the only honest comparisons keep analog on the air:

- **Scenario B — the SCA baseband (~36 kHz).** Put the waveform in the subsidiary-carrier
  space above RDS (≈ 60–96 kHz of the FM multiplex). Rides alongside analog FM, costs the
  station nothing but a little modulation budget. This is the niche analog SCA and HD2/HD3
  multicast already compete in.
- **Scenario C — IBOC's own RF footprint (~140 kHz).** The apples-to-apples what-if: give
  the waveform the *same* spectrum hybrid IBOC's digital sidebands already occupy
  (≈ 70 kHz × 2 flanking the analog), and see what a 2025 modem does with it versus a 2002 one.

This is a hobby project, not a product. The point is the engineering question: *how far
ahead of HD Radio is a modern stack, and by how much?*

## Method (what's measured vs. cited)

- **Required Es/N0 per modcod** — the modem's `computeThreshold`, now **measurement-validated**:
  `benchmark_waterfall` sweeps the full ladder over AWGN, and every rung agrees with the model
  to ~1 dB (modeled / measured closing SNR: QPSK½ 2.2/2.2, QPSK¾ 5.2/5.2, 16-QAM½ 7.0/8.0,
  16-QAM¾ 11.0/11.0, 64-QAM⅔ 14.4/14.4, 64-QAM⅚ 17.8/17.8, 256-QAM¾ 21.0/21.0). On the
  Watterson-HF fading channel the high-order modcods are deep-fade-limited and close much later
  (or not within +8 dB) — physically expected, and a reminder that the SNR column is the AWGN
  best case.
- **Net throughput** — `ComputedParams::net_bitrate` with the occupied bandwidth set to each
  scenario's width. This already accounts for guard bands, pilots (1/8), cyclic prefix (1/8),
  the LDPC code rate, and the Reed-Solomon outer code. Generated by `tools/scorecard.cpp`.
- **HD Radio figures** — published/approximate: hybrid IBOC carries ~96 kbps (HD1, MP1
  service mode) up to ~150 kbps in extended modes, in ~140 kHz of flanking digital sidebands,
  with the proprietary HDC codec (HE-AAC-derived) and convolutional FEC. Treat as ±.

The adaptivity is the whole point: one waveform spans **0.31 → 3.86 bits/Hz**, picking the
densest modcod the channel will carry and degrading gracefully — never a cliff.

## Scenario B — alongside analog FM, in 36 kHz of SCA

A real SCA subcarrier sits high in the FM baseband, where the FM demodulator's triangular
noise is worst and injection is limited (~10%), so the *realistic* operating window is the
robust end: **QPSK to 16-QAM, ~22–44 kbps for solid coverage**, with 64-QAM (~92 kbps) only
on strong local signals.

Even that floor is a decisive win over what shares this space today:

- **Analog SCA**: one ~5 kHz mono channel. Groundwave delivers a *stereo* program (or several)
  with FEC and graceful loss recovery in the same slot.
- **HD2/HD3 multicast**: HDC at ~24–48 kbps — exactly where it turns swishy and underwater.
  Groundwave runs **Opus stereo at 44–92 kbps**, which is cleanly better at equal-or-lower rate,
  **and DRED rides through dropouts instead of muting**.

…all of it riding alongside the analog signal, costing only a sliver of modulation budget,
and **royalty-free**.

## Scenario C — the same RF as hybrid IBOC, 140 kHz

This is the fair fight. In the identical spectrum HD Radio's digital sidebands occupy:

- **At IBOC-class robustness (QPSK):** Groundwave carries **86–135 kbps** — i.e. it *matches*
  IBOC's entire 96–150 kbps payload, but with modern LDPC that closes ~2 dB lower, so the
  *same bitrate reaches further* (more coverage area).
- **When the channel allows:** 16-QAM gives **172 kbps**, 64-QAM **356 kbps**, 256-QAM
  **540 kbps** — roughly **2–5× IBOC's payload in the same Hz**, because IBOC is locked to a
  fixed, robust, low-order mode and cannot trade margin for throughput.

Put plainly: same spectrum, you either get **IBOC's bitrate with more coverage**, or **2–5×
the bitrate** — your choice, per-second, automatically.

## Audio

| | HD Radio | Groundwave |
|---|---|---|
| Codec | HDC (proprietary, HE-AAC-derived) | **Opus 1.5** (open, royalty-free) |
| HD1 / primary | ~96 kbps | transparent-class at 96–128 kbps using only QPSK/16-QAM |
| Multicast / low rate | ~24–48 kbps, audible SBR artifacts | Opus at 32–64 kbps, markedly cleaner |
| Loss behaviour | blend-to-analog (hybrid) / dropout | **DRED** neural recovery (~up to 1 s) |

At every bitrate that matters, Opus meets or beats HDC — and pays no license.

## The four-axis scorecard

| Axis | HD Radio (hybrid IBOC) | Groundwave | Verdict |
|---|---|---|---|
| **Spectral use** | fixed ~0.7–1.1 net bits/Hz | adaptive 0.31–3.86 bits/Hz | Groundwave: more reach at iso-rate, 2–5× rate when SNR allows |
| **FEC / coding** | convolutional (2002-era) | QC-LDPC + RS + ORBGRAND/BICM-ID | Groundwave: ~2 dB+ coding gain |
| **Audio** | HDC, licensed | Opus 1.5 + DRED, open | Groundwave, clearly |
| **Adaptation** | fixed service modes | AMC / VCM / hierarchical UEP | Groundwave: graceful, no cliff |
| **Licensing** | Xperi royalties (broadcaster + receiver) | royalty-free top to bottom | Groundwave — the whole reason it exists |

## Honest caveats

- **This is simulation.** Every number above comes from validated link-budget math and
  Monte-Carlo BER/FER sims, not off-air hardware. The decisive unknowns (real SCA SNR with
  the high-baseband noise penalty, FM multipath, mask/crosstalk on a real exciter, a receiver)
  are untested. The comparison is "modern modem on paper" vs. "deployed 2002 system" — and
  on paper the modern modem *should* win; that's almost the point.
- The **256-QAM ceiling won't happen on a real SCA** — it's noise-limited; treat the
  high-modcod rows as the strong-signal headroom, not the operating point.
- HD Radio is **complete, FCC-authorized, and has receivers**. Groundwave has none of that.
  This scorecard is about the *waveform and codec*, where it is genuinely years ahead — not
  about deployment readiness, where IBOC obviously leads.

## Reproduce

```
cmake --build build --config Release --target scorecard benchmark_waterfall
./build/Release/scorecard.exe          # the throughput/SNR ladder
./build/Release/benchmark_waterfall.exe # the BER/FER waterfalls (AWGN + Watterson-HF)
```
